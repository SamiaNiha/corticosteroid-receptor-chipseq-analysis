# corticosteroid-receptor-chipseq-analysis
ChIP-seq differential binding analysis of glucocorticoid and mineralocorticoid receptor occupancy in rat hippocampus using DiffBind, DESeq2 and Bioconductor.


# ChIP-seq Analysis of Corticosteroid Receptor Binding Dynamics Across Circadian and Stress States in the Rat Hippocampus

## About This Project

This repository contains the code, figures, presentation materials and research report from my MSc Bioinformatics dissertation project.

Using publicly available ChIP-seq datasets generated by Mifsud et al. (2021), I re-analysed glucocorticoid receptor (GR) and mineralocorticoid receptor (MR) binding patterns in the rat hippocampus to investigate how circadian rhythm and acute stress influence receptor-genome interactions.

The project involved developing an R-based bioinformatics workflow incorporating DiffBind, DESeq2, ChIPseeker and pathway enrichment analysis to identify differential receptor occupancy and explore the biological processes associated with stress-responsive genomic regions.

## Research Question

Does acute forced-swim stress produce a distinct receptor binding signature when the hippocampus is already exposed to elevated glucocorticoid levels during the circadian peak?

To address this question, three physiological conditions were compared:

* BLAM – Baseline Morning
* BLPM – Baseline Afternoon (Circadian Peak)
* FS – Forced Swim Stress

The primary focus of the project was to determine whether stress induces additional receptor recruitment on a genomic landscape that is already engaged by the normal circadian rise in glucocorticoids.

## Data Source

This project was based on the re-analysis of publicly available ChIP-seq datasets generated by:

**Mifsud KR, Kennedy CLM, Salatino S, Sharma E, Price EM, Haque SN, et al. (2021). Distinct regulation of hippocampal neuroplasticity and ciliary genes by corticosteroid receptors. Nature Communications, 12, 4737.**

While the original study characterised corticosteroid receptor binding across physiological states, this project applied an independent bioinformatics workflow to further investigate differential receptor occupancy, chromatin context and pathway enrichment across circadian and stress conditions.

## Bioinformatics Workflow

The analysis workflow included:

1. ChIP-seq peak calling using MACS2
2. Consensus peak generation
3. Differential binding analysis using DiffBind
4. Statistical modelling using DESeq2
5. Peak annotation using ChIPseeker
6. Functional pathway analysis using Ingenuity Pathway Analysis (IPA)
7. Data visualisation using PCA plots, heatmaps, volcano plots and Venn diagrams

## Key Findings

* Glucocorticoid receptor (GR) binding showed the strongest response to acute stress, with clear differential occupancy observed between stress and circadian peak conditions.
* Mineralocorticoid receptor (MR) binding was comparatively stable, suggesting a more consistent role across physiological states.
* The histone modification H3K9acS10p showed relatively limited redistribution at the 30-minute stress time point.
* Pathway analysis identified enrichment of signalling networks involved in neuroplasticity, circadian regulation, stress adaptation and neuronal signalling.
* Key pathways included CREB signalling, ERK/MAPK signalling, PI3K/AKT signalling, GPCR signalling and axonal guidance pathways.

Overall, the findings support a model in which circadian phase establishes a permissive genomic environment that is selectively modified by acute stress through targeted glucocorticoid receptor recruitment.
